HTML / CSS 講師

李建杭 (Amos Lee)

從視覺設計轉職前端的斜槓開發者，活躍於臺灣各大技術研討會，熟稔HTML & CSS、Bootstrap、RWD、CSS3等，為國內知名前端技術講師。

2018 年成為Youtuber，致力於使用簡單易懂的方式讓新手能夠輕鬆學習前端開發技能。

iT 邦幫忙鐵人賽連續三屆冠軍、連續兩屆優選，於 2020 出版「金魚都能懂的 CSS 選取器：金魚都能懂了你還怕學不會嗎（iT邦幫忙鐵人賽系列書）」一書，除了熱衷於牛排的熟成，目前日益精進於釣魚技能。
